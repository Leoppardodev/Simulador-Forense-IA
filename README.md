# 📚 Simulador de Questões para Concursos (Powered by Leoppardo)
**Versão Disponível:** Polícia Científica (Perito Criminal PCPB e PCPE).

Este é um aplicativo desktop (MVP) desenvolvido para gerar questões inéditas e comentadas para concursos públicos, utilizando Inteligência Artificial. Em vez de depender de bancos de questões desatualizados, o aplicativo atua como um examinador sênior, criando cenários práticos e desafiadores baseados nos editais mais recentes.

Desenvolvido por **Leonardo Aguiar**.

## 🚀 Como Instalar e Rodar o Aplicativo

Você **não** precisa saber programação ou ter Python instalado no seu computador. O aplicativo já vem empacotado e pronto para uso no Windows.

1. Baixe o arquivo `.zip` mais recente na aba [Releases](#) (Substitua pelo seu link).
2. Clique com o botão direito no arquivo baixado e selecione **"Extrair Tudo..."**.
3. Abra a pasta extraída e dê um duplo clique no arquivo executável (`.exe`).
4. **⚠️ Aviso sobre Antivírus:** Como este é um software independente e protegido (ofuscado para segurança do código), o Windows Defender ou seu antivírus podem exibir um alerta de "Software Desconhecido" na primeira execução. Isso é um falso positivo comum. Basta clicar em **"Mais informações"** e depois em **"Executar assim mesmo"**.

## 🔑 Como Gerar suas Chaves API (Gratuitas)

Para que o aplicativo consiga gerar as questões, ele precisa se comunicar com os motores de Inteligência Artificial. Para isso, você precisará de uma "Chave de Acesso" (API Key) gratuita. 

Siga o passo a passo abaixo para gerar as suas em menos de 2 minutos:

### Opção 1: Chave do Google Gemini (Principal e Recomendada)
O motor do Google é extremamente rápido e possui uma cota diária gratuita muito generosa.

1. Acesse o site oficial de desenvolvedores do Google: [Google AI Studio](https://aistudio.google.com/)
2. Faça login com a sua conta do Gmail padrão.
3. No menu lateral esquerdo, clique em **"Get API key"** (Obter chave de API).
4. Clique no botão azul **"Create API key"** e escolha a opção para criar em um novo projeto (*"Create API key in new project"*).
5. O site vai gerar um código longo (começando geralmente com `AIza...`). **Copie esse código.**

### Opção 2: Chave do OpenRouter (Redundância/Opcional)
O OpenRouter é um roteador que permite acessar várias IAs diferentes. É ótimo ter essa chave cadastrada caso o Google fique fora do ar.

1. Acesse o site: [OpenRouter.ai](https://openrouter.ai/)
2. Faça login (pode usar a sua conta do Google para facilitar).
3. No canto superior direito, clique na sua foto de perfil e vá em **"Keys"**.
4. Clique em **"Create Key"**, dê um nome qualquer (ex: "Simulador Concurso") e clique em criar.
5. **Copie o código gerado** (começa com `sk-or-v1...`). *Atenção: Ele só aparece uma vez!*

## ⚙️ Configurando o Aplicativo

1. Abra o Simulador de Concursos no seu computador.
2. No menu lateral, clique em **"⚙️ Configurações (API)"**.
3. Cole a chave do Google no campo "Chave Principal (Gemini)".
4. (Opcional) Cole a chave do OpenRouter no campo "Provedor Redundância".
5. Pronto! O aplicativo salvará suas chaves automaticamente. Clique em **"🏠 Dashboard"** no menu lateral, escolha a disciplina e comece a gerar suas questões!
